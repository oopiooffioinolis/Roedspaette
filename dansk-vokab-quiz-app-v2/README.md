# Dansk Vokab — Quiz App

Spaced-repetition flashcards for the vocab you collect with the Dansk Vokab Chrome extension. Static app, no server: it reads your Excel set files from your private GitHub repo and writes review progress back into their hidden columns.

**Setup:** see QUIZ_APP_GUIDE.md (or the chat where this was built). Short version: host this folder on GitHub Pages, open the URL, paste your fine-grained token, study. On iPhone: Safari → Share → Add to Home Screen.

Privacy: this repo contains only app code. Your token is entered on-device and stored in that browser only; your vocabulary stays in your private repo.
